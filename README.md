# 🟢 Academic Truth: ChatGPT Reference Validator

Real-time fact-checking & auto-repair engine for ChatGPT citations. Fight AI hallucinations and "Frankenstein references" instantly.

---

## 🚀 How to Install (Try it instantly!)

Since this is a developer preview, you can install it manually in **1 minute**:

1. Click the green **`Code`** button at the top of this GitHub page and select **`Download ZIP`**.
2. Extract (unzip) the downloaded folder on your computer.
3. Open Google Chrome and navigate to: `chrome://extensions/`
4. Toggle **ON** the **"Developer mode"** in the top-right corner.
5. Click **"Load unpacked"** in the top-left corner and select the extracted folder.
6. Refresh your ChatGPT page and start researching!

---

## 🛠️ Key Features

- **Traffic Light Badges:** Instantly flags citations as `GREEN` (Verified), `YELLOW` (Frankenstein ref), or `RED` (Fake).
- **Auto-Repair Engine:** If a reference is hallucinated, our tool automatically reverse-searches Semantic Scholar to suggest the **actual real paper** you need.
- **Clean BibTeX Export:** Download only the verified `GREEN` references in one click for Zotero or Mendeley.
